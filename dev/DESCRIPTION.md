### **GPTwidget: Premium ChatGPT Plasmoid for KDE Plasma 6**

**GPTwidget** is a lightweight, privacy-hardened ChatGPT widget designed specifically for the KDE Plasma desktop. It provides a dedicated workspace for your AI interactions, seamlessly integrated into your workflow.

#### **Key Features:**
*   🚀 **Dedicated Experience**: Hardcoded for ChatGPT to ensure maximum stability and a focused UI.
*   🛡️ **Advanced Stability Layer**:
    *   **Crash Recovery**: Auto-restarts the browser engine if it terminates unexpectedly.
    *   **Load Watchdog**: Automatically recovers from "hung" or "stale" page loads.
*   🔒 **Privacy First**: Permanently disabled webcam, screensharing, and notification permissions.
*   🌗 **Breeze Integration**: Automatically syncs with your Plasma Dark/Light theme for a native look.
*   ⌨️ **Productivity Boosts**: `Ctrl + F` search, automatic input focus, and external links opening in your default browser.
*   📥 **Integrated Download Manager**: Handles file outputs with path persistence.
